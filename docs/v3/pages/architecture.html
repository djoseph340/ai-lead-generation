Architecture & Nodes

12-Node Workflow

v3 consists of 12 specialized nodes organized into functional groups.

Workflow Stages

1. Webhook: Lead Intake

Entry point for lead submissions with authentication

2. Validate & Sanitize

Data validation and input sanitization

3. Apollo Company Enrichment

Real-time company data enrichment via Apollo API

4. Apify Web Scraper

Automated web research and competitive intelligence

5. Claude: Intent Classification

AI analysis of lead intent and urgency signals

6. RAG Knowledge Base Lookup

Context-aware retrieval from proprietary knowledge base

7. Advanced Multi-Factor Scoring

10+ weighted factors for comprehensive scoring

8. 5-Tier Assignment

Qualification tier assignment with reasoning

9. Append to Google Sheets

Store qualified leads in Google Sheets

10. Slack Notification

Real-time Slack alerts for high-value leads

11. CRM Sync

Live synchronization to HubSpot/Salesforce

12. Return Success Response

Confirmation response to webhook source

Architecture Overview
  • Total Nodes: 12
  • AI Agents: 1 (Claude)
  • APIs: Apollo, Apify, RAG
  • Outputs: Sheets, Slack, CRM
  • Processing Time: 3-5 seconds
