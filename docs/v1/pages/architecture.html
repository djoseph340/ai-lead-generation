Architecture

System Flow

Components

Webhook (Entry Point)

Receives POST requests from form builders. Endpoint: /lead-intake

Code Node (Extract + Score)

JavaScript that extracts fields dynamically and calculates lead score using rule engine.

Conditional Router

IF/ELSE node that branches based on score threshold (80 points).

Google Sheets Output

Two tabs: Leads (qualified) and Unqualified (with tiers and reasons).

Implementation

Built as an n8n workflow. Components interact synchronously with no queue or delays.

What n8n Handles
  • Webhook trigger
  • Node orchestration
  • Google Sheets API calls
  • Error logging
What We Code
  • Scoring rules
  • Field extraction
  • Tier classification
