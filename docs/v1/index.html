Lead Qualification Assistant

Automated lead intake, scoring, and routing via n8n

What This Does

This project is an automated workflow that qualifies inbound leads. It receives form submissions, scores them based on defined rules, and routes them to appropriate storage buckets based on qualification level.

The system uses deterministic rules—not machine learning—to ensure scores are explainable and can be tuned without retraining.

Quick Facts

  • ✓ Rule-based, explainable scoring
  • ✓ Dynamic field extraction from forms
  • ✓ Two-tier classification system
  • ✓ Built in n8n
  • ✓ Uses Google Sheets for storage

Known Limitations

  • • No learning from past outcomes
  • • No external enrichment APIs
  • • Depends on consistent form labels
  • • No CRM integration yet
  • • Rules-only (no ML component)
