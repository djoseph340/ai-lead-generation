Frequently Asked Questions

How do I customize the scoring factors?

Edit the "Advanced Multi-Factor Scoring" node in n8n. Adjust the weights for each dimension or add new factors by modifying the JavaScript code.

Can I integrate with HubSpot or Salesforce?

Yes. Add HTTP Request nodes after the Compile Lead Profile step to sync qualified leads to your CRM.

What happens to invalid leads?

They are rejected at the Validation Gate and a 400 error is returned with detailed error messages.

How many leads can this handle?

v2 can process 1000+ leads per day. For higher volumes, consider upgrading to v3 or scaling your n8n infrastructure.

Is my data secure?

All credentials are stored encrypted in n8n. No lead data is logged unless explicitly configured. Enable audit logging for compliance.

Can I modify the tier assignment logic?

Yes. Edit the "Intelligent Tier & Status Assignment" node to change the score thresholds and tier definitions.
